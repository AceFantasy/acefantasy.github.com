1. 团队介绍

数据智能团队，担当着 IIC 及达摩院数据解决方案建设，以及数据机器人等智能型数据产品的创新和研发，坚持数据和技术驱动，坚持推动实现业务数据化、数据智能化、智能业务化的数据之路。

2. 数据技术专家

职责：

  • 数据智能领域开拓者，引领数据智能创新方向。
  • 充分结合阿里大数据和人工智能技术优势，为阿里生态孵化数据创新业务。
  • 坚持数据和技术驱动，实现业务数据化、数据智能化、智能业务化的数据之路。

要求：

  • 至少 5 年以上带团队经验。
  • 对数据仓库、数据分析、数据洞察/预测/决策类算法、数据化运营体系等有丰富经验和成功案例。
  • 对大数据、人工智能、以及最近崛起的数据智能领域的主流技术方案有深刻理解；对未来发展趋势有自己的见解。
  • 具备足够的数据创新意识和商业敏感度，能着手推动新兴数据产业发展。

3. 大数据研发专家

职责：

  • 参与打造达摩院机器智能以及智能服务部的数据架构设计与研发。
  • 负责达摩院层面重点项目中的数据开发和体系建设，通过数据技术给业务带来增量。
  • 为团队引入创新的技术和方案，用创新的思路解决问题，能对现存或未来系统进行宏观的思考，规划形成统一的框架、平台或组件。

要求：

  • 具有丰富的数据建模实践经验及分布式平台数据处理相关工作经验。
  • 具备扎实的理论基础和数据技术，具备大数据平台实时和离线 ETL 开发能力，参与或负责过线上业务的数据仓库建设。
  • 技术或业务理解上有亮点，自我学习，对新技术敏感。
  • 良好的思维逻辑性、语言表达能力；有较好英语口语能力优先。
  • 具备一定的 JAVA 或 Python 的开发能力，熟悉机器学习算法能力优先。

4. Java 技术专家

职责：

  • 数据类产品的架构设计及编码工作。
  • 人工智能相关产品和工具的探索与开发。
  • 善于学习研究业界前沿技术，并迅速转化为项目生产力。

要求：

  • 3 年以上 Java 开发经验。
  • 精通 Java 开发语言，对 Java 底层原理有一定理解，了解多线程、非阻塞、一致性等后端分布式系统常用技术。
  • 有分布式计算、中间件、高性能服务端等开发经验。
  • 有基于开源项目进行二次开发的经验，基于 Hadoop 生态圈（Hive，HBase，Spark，Storm, Presto等相关技术）开发数据服务系统的经验，基于 Spring Boot 或者其他微服务和轻量级容器技术开发的经验。
  • 有高并发高可有环境开发经验，熟悉负载均衡、FailOver 等处理方案。
  • 对数据和人工智能结合的工程技术有很强的兴趣。
  • 有私有云、公有云经验者优先考虑。

5. 前端开发专家

职责：

  • 数据类场景，产品设计，技术架构设计及编码工作。

要求：

  • 5 年以上工作经验。
  • 有带团队，独立设计并完成较复杂场景的开发任务的经验。
  • 熟悉 React 或其它当前流行的框架。
  • 有数据类领域经验（包括不限于，数据可视化，数据仓库工具，可视化编辑器等）优先。
  • 有后端应用开发经验优先。

6. 算法专家

职责：

  • 数据洞察：对数据进行异常检测和智能下钻，找到数据异常归因，并生成洞察报告；
  • 时序预测：通过对时间序列数据进行建模和预测；
  • 调度优化：实时对大量用户与客服进行最优化匹配，对全局资源进行资源最优化的调度。

要求：

  • 具备丰富的机器学习、运筹优化或强化学习相关领域经验。
  • 熟悉主流深度学习框架（tensorFlow/mxnet/pytorch）及应用经验。
  • 具备良好的逻辑思维、对数据敏感、责任心、有激情、团队合作、沟通能力佳。

7. 数据分析专家

职责：

  • 理解业务构建业务的分析体系和洞察模型，及时准确地为业务线提供有洞察力的数据分析结论和策略建议，向业务团队反向提出优化方向及优化策略。
  • 基于外部市场环境和内部业务的监控，主动挖掘有价值的商业分析专题。通过专题分析，为公司商业决策、产品方向、销售及运营策略、用户增长等提供数据支持。
  • 沉淀分析思路与框架，产品化，支持数据化运营。

要求：

  • 计算机、统计、数学等相关专业本科及以上学历，5年以上工作经验，3年以上BI或数据分析相关经验。
  • 熟悉数据分析的基本概念和方法，熟练使用技术手段进行数据分析和挖掘。
  • 具有良好的商业敏感性、逻辑思维能力、数据敏感性，能够承受一定的工作压力，学习能力强。
  • 具备优秀的沟通协调能力和团队合作精神，有高度的责任感，有上进心和主动性，有较强的抗压能力。

8. 解决方案架构师

职责：

  • 立足阿里生态的全域数据，应用大数据、机器学习、统计分析和数据可视化等技术，探索和发展数据智能之路。
  • 挖掘阿里经济体、外部客户的业务需求，甄别业务场景和商业价值，通过业务场景创新沉淀数据解决方案框架和方法论。
  • 驱动方案快速落地并实现规模化；驱动数据资产的沉淀，打造数据赋能能力；驱动数据产品创新和升级，打造数据商业化能力。

要求：

  • 在一个或多个领域有扎实的理论基础和实战经验：统计分析、数据挖掘、机器学习、自然语言处理等。
  • 具备优秀数据挖掘和分析能力，熟练使用Python/R至少一种语言，具备扎实的数据处理能力，熟练使用SQL。
  • 具有良好的商业敏感度，有项目的成功经验，能够横向协同，跨界整合资源，驱动项目落地与产品化建设。
  • 具有创新思维，学习能力强，具备抗压能力，善于沟通和团队协作，乐于分享。
  • 具有本科以上学历，5 年以上相关工作经验，硕士、博士优先。
